<task type="auto">
  <name>Task 1: Create reset.css and variables.css</name>
  <files>shared/css/reset.css, shared/css/variables.css</files>
  <action>
Create the `shared/css/` directory structure, then create both files:

**reset.css** -- Modern CSS reset based on Josh Comeau 2025 pattern (see RESEARCH.md Pattern 2):
1. Box-sizing border-box on all elements
2. Remove default margins (exclude dialog)
3. Smooth scrolling with prefers-reduced-motion respect
4. Body defaults: min-height 100vh, line-height 1.6, font-smoothing antialiased
5. Media defaults: block display, max-width 100%
6. Form element font inheritance
7. Text overflow-wrap: break-word on p and headings
8. text-wrap: pretty on p, text-wrap: balance on headings (progressive enhancement)
9. Reduced motion: disable all animations/transitions
10. Text size adjust prevention for orientation changes

**variables.css** -- All CSS custom properties in :root (see RESEARCH.md Pattern 1 + Code Examples):

Colors:
- Primary blue family: --color-primary (#005982), --color-primary-light (#0077AD), --color-primary-dark (#003D5C), --color-primary-50 (#E6F2F8), --color-primary-100 (#B3D9EA)
- Accent orange: --color-accent (#E67E22), --color-accent-hover (#D35400), --color-accent-light (#FDEBD0)
- Backgrounds (warm, NOT pure white): --color-bg (#FDFAF6), --color-bg-alt (#F5F0EB), --color-surface (#FFFFFF)
- Text (warm, NOT pure black): --color-text (#2C3E50), --color-text-light (#5D6D7E), --color-text-on-dark (#F8F9FA), --color-text-on-primary (#FFFFFF)
- Semantic: --color-success (#27AE60), --color-border (#E0DCD7)

Typography:
- --font-heading: 'Nunito', 'Segoe UI', Tahoma, sans-serif
- --font-body: 'Open Sans', -apple-system, system-ui, sans-serif
- Fluid type scale using clamp() (400px-1280px viewport range): --fs-sm through --fs-xxl (use exact clamp values from RESEARCH.md)
- --line-height-body: 1.6, --line-height-heading: 1.2, --letter-spacing-heading: -0.02em

Spacing scale (rem-based):
- --space-xs (0.25rem) through --space-4xl (6rem)
- --space-section: clamp(3rem, 2.5rem + 2.5vw, 6rem)

Layout: --container-max (1200px), --container-padding (var(--space-md))
Border radius: --radius-sm (4px) through --radius-full (9999px)
Shadows: --shadow-sm through --shadow-xl
Transitions: --transition-fast (150ms), --transition-base (250ms), --transition-slow (400ms)

Do NOT use @import between CSS files. Each file is independent and loaded via separate link tags in HTML.
  </action>

<task type="auto">
  <name>Task 2: Create base.css and utilities.css</name>
  <files>shared/css/base.css, shared/css/utilities.css</files>
  <action>
**base.css** -- Base element styles that reference variables.css tokens (see RESEARCH.md Code Examples):

Body:
- font-family: var(--font-body), font-size: var(--fs-base), line-height: var(--line-height-body)
- color: var(--color-text), background-color: var(--color-bg)

Headings (h1-h6):
- h1: font-family var(--font-heading), font-size var(--fs-xxl), font-weight 800, line-height var(--line-height-heading), letter-spacing var(--letter-spacing-heading), color var(--color-primary-dark)
- h2: var(--fs-xl), weight 700, color var(--color-primary-dark)
- h3: var(--fs-lg), weight 700, color var(--color-text)
- h4: var(--fs-md), weight 700, color var(--color-text)
- h5: var(--fs-base), weight 700, color var(--color-text)
- h6: var(--fs-sm), weight 700, color var(--color-text-light)
- Add margin-bottom: var(--space-md) to all headings for vertical rhythm

Paragraphs:
- margin-bottom: var(--space-md), max-width: 70ch (readability), font-size: var(--fs-base)

Links:
- color: var(--color-primary), text-decoration-skip-ink: auto
- hover: color var(--color-primary-light)

Lists (ul, ol):
- padding-left: var(--space-xl), margin-bottom: var(--space-md)
- li margin-bottom: var(--space-xs)

**utilities.css** -- Reusable utility classes (see RESEARCH.md Essential Utility Classes):

Layout:
- .container: max-width var(--container-max), margin-inline auto, padding-inline var(--container-padding)
- .section: padding-block var(--space-section)
- .section--alt: background-color var(--color-bg-alt)
- .section--dark: background-color var(--color-primary-dark), color var(--color-text-on-dark)

Grid (mobile-first):
- .grid: display grid, gap var(--space-lg), grid-template-columns 1fr (mobile default)
- @media (min-width: 768px): .grid--2 gets 2 columns, .grid--3 gets 2 columns
- @media (min-width: 1024px): .grid--3 gets 3 columns

Accessibility:
- .sr-only: clip-rect pattern (position absolute, 1px x 1px, overflow hidden, clip rect(0,0,0,0), whitespace nowrap)

Text:
- .text-center, .text-accent, .text-primary

Buttons:
- .btn: inline-flex, centered, gap var(--space-sm), padding var(--space-sm) var(--space-xl), font-family var(--font-heading), weight 700, border-radius var(--radius-md), cursor pointer, transitions on background-color + transform + box-shadow
- .btn:hover: translateY(-2px), box-shadow var(--shadow-md)
- .btn--primary: background var(--color-accent), color white; hover: var(--color-accent-hover)
- .btn--outline: transparent bg, color var(--color-primary), border 2px solid var(--color-primary); hover: filled primary bg, white text

Responsive container padding:
- @media (min-width: 768px): update --container-padding to var(--space-xl) in :root

Cards (basic):
- .card: background var(--color-surface), border-radius var(--radius-md), padding var(--space-xl), box-shadow var(--shadow-sm), transition on box-shadow + transform
- .card:hover: box-shadow var(--shadow-md), translateY(-2px)
